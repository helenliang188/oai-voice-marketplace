# OAI Voice Marketplace

Codex plugin marketplace for `oai-voice`, a writing skill that rewrites communications so they are direct, conversational, evidence-led, and values-grounded.

## Install

Add the public marketplace:

```bash
codex plugin marketplace add helenliang188/oai-voice-marketplace
codex plugin add oai-voice@oai-voice-marketplace
```

Then start a new Codex thread and invoke:

```text
Use $oai-voice to rewrite this paragraph: ...
```
